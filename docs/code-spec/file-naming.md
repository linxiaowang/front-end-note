# 项目文件命名

## 项目名
全部采用小写方式， 以短横线分隔。 例：`my-project-name`

## 目录名
参照项目命名规则，有复数结构时，要采用复数命名法。例：docs、assets、components、directives、mixins、utils、views。

## 图像文件名
全部采用小写方式， 优先选择单个单词命名，多个单词命名以下划线分隔。

```
banner_sina.gif
menu_img.png
```

## HTML文件名
全部采用小写方式， 优先选择单个单词命名，多个单词命名以下划线分隔。

```
banner_sina.html
menu_img.html
```

## CSS文件名
全部采用小写方式， 优先选择单个单词命名，多个单词命名以短横线分隔

```
base.css
date-picker.scss
```
## JavaScript 文件名
全部采用小写方式， 优先选择单个单词命名，多个单词命名以短横线分隔
```
index.js
plugin.js
date-util.js
```
