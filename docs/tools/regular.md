# 常用正则表达式

## 手机号

```javascript
/^((13[0-9])|(14[0-9])|(15[0-9])|(16[0-9])|(17[0-9])|(18[0-9])|(19[0-9]))\d{8}$/
```

## 邮箱

```javascript
/^\w+([-+.]\w+)*@\w+([-.]\w+)*\.\w+([-.]\w+)*$/;
```
## 密码

```javascript
/^.*(?=.{8,20})(?=.*[a-zA-Z])(?=.*\d)(?=.*[\p{P}]).*$/
```
## IP

```javascript
/^((?=.{3,255}$)[a-zA-Z0-9][-a-zA-Z0-9]{0,62}(\.[a-zA-Z0-9][-a-zA-Z0-9]{0,62})+)(:\d{2,5})?$/
```
